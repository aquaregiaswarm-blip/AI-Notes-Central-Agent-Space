## Pocket Consigliere V2 Features

### Meeting Quality Checklist System
- [ ] **Pre-defined meeting checklists** based on meeting type (discovery, qualification, follow-up, close)
- [ ] **BANT qualification tracker** (Budget, Authority, Need, Timing)
  - Real-time prompts: "Have we identified budget?" "Who is the decision-maker?"
  - Visual checklist in UI (✓ covered, ⚠ partially covered, ✗ not discussed)
- [ ] **Standard sales/consulting must-dos:**
  - Confirm next steps
  - Set follow-up timeline
  - Identify decision criteria
  - Surface objections/concerns
  - Capture success metrics
- [ ] **Customizable checklists** per client or engagement type
- [ ] **Post-meeting completeness score** ("You covered 7/10 critical topics")
- [ ] **LLM auto-detection** of checklist items from transcript (flag if missed)
- [ ] **Historical tracking:** Did we get BANT in first call? Second call?

**Use case:** Ensure consultants and sales reps don't miss critical qualification or next-step questions during live calls.
